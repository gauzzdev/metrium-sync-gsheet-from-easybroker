Metrium | Utilidad de sincronización

Esta herramienta sincroniza todas las propiedades de Easybroker (cuenta Metrium) con el estatus seleccionado a una hoja de Google Sheets, agregando los datos al final, en el formato necesario para que Meta actualice sus catálogos.

Importante: Para que esta herramienta pueda modificar la hoja de cálculo de Google Sheets, debes compartirla con permisos de edición con el siguiente correo de servicio:

development-gauzz@sync-gsheet-from-easybroker.iam.gserviceaccount.com

¿Cómo obtener el ID de Google Sheets? Abre tu hoja de cálculo en Google Sheets y copia la parte del URL que se resalta aquí:


https://docs.google.com/spreadsheets/d/1_hsDhY7OFZT8YsNLAWlngfLMvFRzTuB1zsccjmqm1_A/edit?gid=0#gid=0

Nota: Mantén presionado Ctrl (Windows) o Cmd (Mac) para seleccionar múltiples estatus. Se obtendrán todas las propiedades con los estatus seleccionados.

Nota: Si no seleccionas ningún tipo, se obtendrán todos los tipos de propiedades. Usa Ctrl (Windows) o Cmd (Mac) para selección múltiple.
